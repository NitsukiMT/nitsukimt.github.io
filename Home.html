WELCOME TO NITGAMES
  • Trang web game NITGames được xây dựng nhằm mục tiêu thực hành và áp dụng các kiến thức đã học trong học phần Thiết kế Web, bao gồm HTML, CSS và JavaScript. Giao diện được thiết kế thân thiện với người dùng, dễ sử dụng, phù hợp với mọi đối tượng yêu thích trò chơi điện tử.
  • Trang web bao gồm các chức năng cơ bản như đăng nhập, đăng ký, giới thiệu game và nút mua hàng minh họa. Giao diện sử dụng hiệu ứng làm mờ nền (glassmorphism) hiện đại, kết hợp với icon và bố cục responsive.
  • Qua việc xây dựng trang web này, em đã củng cố được kỹ năng lập trình giao diện, cách tổ chức file HTML – CSS – JS, xử lý tương tác người dùng và kiểm tra dữ liệu đầu vào. Đây là nền tảng quan trọng cho việc phát triển các ứng dụng web hoàn chỉnh trong tương lai.
